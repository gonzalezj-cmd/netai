Base de datos: PostgreSQL
Lenguaje: Python
Framework API: FastAPI


================================================
OBJETIVO DEL SISTEMA
================================================

Softmikro es un sistema de monitoreo para redes ISP basado en MikroTik.

Permite:

- monitorear sesiones PPPoE
- monitorear tráfico de interfaces
- detectar caídas de PPPoE
- detectar saturación de VLAN
- detectar revendedores NAT
- crear dashboards NOC
- almacenar histórico de red

ejecutar API
